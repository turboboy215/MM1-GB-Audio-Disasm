# MM1-GB-Audio-Disasm
Mega Man: Dr. Wily's Revenge (Game Boy) audio disassembly

A disassembly of the sound engine and data from Mega Man: Dr. Wily's Revenge for Game Boy, fully labeled and commented. Can be built with RGBDS; the sound data of the resulting ROM is verified to match the sound data from the original ROM exactly. The resulting ROM includes a custom player where all the music and sound effects can be heard.
A text file is also included which documents how the music, sound effect, and instrument data is structured for this game as well as other games using versions of the same sound engine.
